## 설치 및 실행

배포 링크가 동작하지 않을 시, local에서 프로젝트를 실행하는 방법을 작성할 영역입니다. 

### Prerequisites

1. npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Repo를 clone한다
   ```sh
   git clone https://github.com/gml9812/wanted-frontend-pre-on-boarding-mission.git
   ```
2. NPM pakage들을 설치한다
   ```sh
   npm install
   ```
3. localhost에서 프로젝트를 실행한다
   ```sh
   npm start
   ```
